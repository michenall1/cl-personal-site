a keystone based cms website
more information about keystone: http://keystonejs.com/

## develop
#### start mongodb:
- `$ ./db/mongod`

#### repair mongodb:
- `$ mongod --dbpath ./db/data --repair`

#### compile client:
- `$ npm run compile`

#### start keystone:
- `$ cd circle-lu`
- `$ node keystone`

#### visit http://localhost:8080
